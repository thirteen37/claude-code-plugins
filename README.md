# thirteen37-plugins

Personal Claude Code plugins marketplace by Yu-Xi Lim.

## Installation

Add this marketplace to Claude Code:

```bash
claude /install marketplace github:thirteen37/thirteen37-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [workbranch](https://github.com/thirteen37/workbranch) | Git worktree manager for isolated development |

## Installing Plugins

After adding the marketplace, install plugins with:

```bash
claude /install workbranch
```
